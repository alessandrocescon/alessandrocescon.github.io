Un articolo speciale per questa nostra avventura nel mondo del puro javascript

Dedichiamo uno spazio tutto speciale per inaugurare questa nuova avventura estrema. Creare un cms file based utilizzando solo codice lato client (javascript) per la navigazione e la ricerca. Tutto il sistema si basa su un array json in cui mappare i contenuti ed indicare direttamente le parole chiave di ricerca.
